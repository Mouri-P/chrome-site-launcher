# Default Images

Place your custom background images in this folder.

## Supported Formats
- JPG/JPEG
- PNG
- WebP
- SVG

## Usage
1. Add your image files to this folder
2. Open the theme settings (click the theme button)
3. Select "Image" as background type
4. Choose from default images or upload your own

## Note
Images uploaded through the theme settings are stored in Chrome's local storage as data URLs. Images placed directly in this folder can be referenced by their path in the theme settings.
