Privacy Policy

Last Updated:

TL;DR: Site Launcher stores all your data locally on your device. We don't collect, transmit, or share any of your personal information.

1. Information We Collect

We don't collect any information. Site Launcher operates entirely on your local device.

2. Data Storage

All data is stored locally on your device using Chrome's local storage API:

  • Your Sites: Website names, URLs, and custom icons you add are stored locally in your browser
  • Theme Preferences: Your background color and image choices are stored locally
  • Custom Images: Any images you upload are converted to data URLs and stored locally

This data is:

  • Never transmitted to any server
  • Never shared with third parties
  • Only accessible on your device
  • Synced across your devices only if you have Chrome Sync enabled (this is handled by Chrome, not by us)

3. External Services

Site Launcher uses the following external services:

  • Google Favicon Service: To fetch website favicons. We only send the domain name (e.g., "google.com") to Google's service. No personal information is transmitted.

4. Permissions

Site Launcher requests the following permissions:

  • storage: To save your sites and preferences locally
  • tabs: To get the current page information when you add a site via the popup
  • activeTab: To access the current page title and URL when adding sites
  • topSites: To import your frequently visited sites (only when you explicitly click the import button)
  • scripting: To inject user-defined scripts into websites (only when you explicitly configure scripts for specific sites)
  • host_permissions (): This broad permission is required to:
    • Fetch favicons from any website you add to your launcher
    • Inject user-defined scripts into websites (only when you explicitly configure scripts for specific sites)
    • Access website content to execute scripts you've configured
    Important: The extension only uses this permission when you explicitly configure scripts for a site. It does not automatically inject scripts or access website content without your explicit configuration.

These permissions are used only for the stated purposes and nothing else.

4.1. Script Injection Feature

Site Launcher includes an advanced feature that allows you to inject custom JavaScript code into specific websites. This feature:

  • Is opt-in only - scripts are only executed when you explicitly configure them for a specific site
  • Requires your explicit action - you must manually add and configure scripts in the extension's settings
  • Includes safety features - you can enable a confirmation popup before scripts execute
  • Is stored locally - all scripts are stored on your device and never transmitted to any server

Warning: Injecting scripts into websites can modify website behavior and may violate website terms of service. Use this feature responsibly and only on websites where you have permission to do so. Site Launcher is not responsible for any consequences resulting from script injection.

5. Third-Party Links

When you click on a site icon, you navigate directly to that website. We don't track which sites you visit or how often you visit them.

6. Data Export and Deletion

You have full control over your data:

  • Export: You can export all your sites as a JSON file at any time
  • Delete: You can delete individual sites or uninstall the extension to remove all data
  • No Account Required: Since we don't collect data, there's no account to delete

7. Children's Privacy

Site Launcher does not knowingly collect any information from anyone, including children under 13. Since we don't collect any data, this is not applicable.

8. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date. We encourage you to review this policy periodically.

9. Open Source

Site Launcher is open source. You can review the source code to verify our privacy claims. The source code is available on GitHub.

10. Contact Us

If you have any questions about this Privacy Policy, please contact us through:

  • GitHub Issues

This privacy policy is effective as of the date listed above and applies to all users of Site Launcher.
